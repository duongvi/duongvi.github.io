Data Visualization with Google Charts

In the last week of the JavaScript course by OneMonth, Google Charts was introduced as a way to include, you guessed it, charts on a website using JavaScript and Google's Charts API. Below is a chart depicting the average IMDb scores of movies from individual years. The data used for that was a dataset from kaggle.com.

You can also hover over the line chart to see what the exact average score was for movies released in that year. Please note that it is not certain, whether the dataset contains all of the movies on IMDb, but my guess is probably not.
