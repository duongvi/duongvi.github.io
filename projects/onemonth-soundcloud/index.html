Soundcloud Miniplayer

In the third week of the OneMonth Learn JavaScript course, another API was introduced, this time from Soundcloud. It included a sending search queries for music on Soundcloud, as well as a embedded player from Soundcloud. Go on, try searching for an artist/song and add it to the playlist on the left, from which you can then play the music. You know you want to. Of course, there is also a button for clearing the left panel.
